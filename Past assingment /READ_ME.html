—————————— To run the models ———————————

1. Run ‘preprocessing.py’ first!!!

——> The results are saved in an Excel file: ‘results.xlsx’


2. Run ‘Model_1B.py’ or ‘Model_2.py’

——> The results are saved in Excel files: ‘Model_1B_results.xls’ or ‘Model_2_results.xls’




—————————— To plot the networks on a map ——————

1. Run ‘Mapping_1B.py’ or ‘Mapping_2.py’


To view results on the map:

2. Open ‘indexy.html’ file in browser (see commented instructions in ‘Mapping_1B.py’ and ‘Mapping_2.py’ files)



—————————— KPIs Calculations ————————————

1. ‘KPIs_Model_1B.py’ calculates the KPIs for the first model (‘Model_1B.py’)

2. The KPIs for the second model (‘Model_2.py’) are already calculated in the last part of the script ‘Model_2.py’
